## üîç Security Awareness Lessons

### How to Identify Spoofed Websites

1. **Check the URL carefully**
   - Real: `https://loginportal.fcps.edu`
   - Fake: `https://fcps-login.com` or `https://fcps.login-portal.com`

2. **Look for HTTPS and valid certificates**
   - Click the padlock icon to verify certificate details

3. **Watch for subtle spelling differences**
   - Typosquatting: `fcsp.edu` instead of `fcps.edu`

4. **Be suspicious of unexpected login requests**
   - Phishing emails often create urgency

5. **Never enter credentials from email links**
   - Always navigate directly to the website
